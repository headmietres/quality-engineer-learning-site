# 8D 报告解决供应商质量问题

## 背景

某电子组装厂收到 IQC 报告：**批次 #2026015** 的 PCB（印刷电路板）在来料检验中发现 **5% 焊盘镀层厚度不足**（规格 25-50 µm，实测 18-22 µm）。

## 8D 过程

### D1：组建团队

| 成员 | 部门 | 角色 |
|------|------|------|
| 质量工程师 | 质量部 | 组长 |
| SQE | 采购部 | 供应商接口 |
| 工艺工程师 | 制造部 | 工艺评估 |
| 供应商代表 | 供应商 | 改善执行 |

### D2：问题描述

**问题陈述：** 批次 #2026015 的 PCB 在 IQC 检测中发现焊盘镀层厚度不足，50 个抽样中有 3 个低于规格下限 25 µm，最低值 18 µm。

### D3：临时遏制措施

| 措施 | 负责 | 完成时间 |
|------|------|----------|
| 隔离该批次 5000 pcs | IQC | 当天 |
| 100% 筛选测量镀层厚度 | 第三方检测 | 3 天 |
| 生产线已使用部分紧急追溯 | 生产部 | 2 天 |

**筛选结果：** 312 pcs 不合格（6.24%），已退回供应商。

### D4：根本原因分析（鱼骨图）

```
人员                         机器
  └─ 操作员未按SOP调整参数     └─ 镀金槽液位传感器故障
  └─ 新员工培训不足            └─ 整流器输出不稳定

                       ┌─── 镀层厚度不足 ───┐

材料                         方法
  └─ 镀金溶液浓度偏低         └─ 电流密度设定错误
  └─ 阳极篮杂质过多           └─ 镀金时间不足
```

**根本原因确认：** 镀金槽 **液位传感器漂移** 导致自动补液系统未及时补充镀液，镀金离子浓度下降，导致沉积速率降低。

**验证：** 校准传感器后发现偏差 +15%，更换传感器后镀液浓度恢复正常。

### D5：永久纠正措施

| 措施 | 负责 | 完成期限 |
|------|------|----------|
| 更换液位传感器（型号升级） | 供应商设备部 | 1 周 |
| 增加液位传感器每日校准点检 | 供应商品质部 | 立即 |
| 增加镀液浓度在线监测（每 2 小时） | 供应商工程部 | 2 周 |
| 增加镀层厚度在线 XRF 检测 | 供应商工程部 | 1 月 |

### D6：措施验证

- ✅ 改善后连续生产 **10 批次**，镀层厚度 Cpk 从 **0.85 提升到 1.92**
- ✅ 后续 3 个月无镀层厚度相关投诉
- ✅ 在线监测系统实现实时报警

### D7：预防再发

- 更新 PFMEA：增加液位传感器失效模式
- 更新控制计划：增加镀液浓度检测频次
- 供应商审核检查表增加传感器校准项

### D8：团队总结

- 从发现问题到关闭共 **21 天**
- 供应商质量评分从 C 级提升到 B 级
- 该 8D 报告作为模板推广到其他供应商

## 关键经验

1. **临时措施要快** — 第一时间隔离不良品，防止流入生产线
2. **根本原因要深** — 不要停在表面原因，要用 5Why 深挖
3. **验证要闭环** — 改善措施必须用数据证明有效
4. **预防很重要** — D7 是防止同类问题再发生的保障
