# SPC 控制图选择指南

选择合适的控制图是 SPC 应用的第一步。本指南帮助您根据数据类型和场景选择正确的控制图。

## 快速选择流程图

```
数据是计数值还是计量值？
│
├── 计量值（连续数据）
│   │
│   ├── 子组大小 n ≥ 2？
│   │   ├── Yes → n ≤ 8？
│   │   │   ├── Yes → X̄-R 图（最常用）
│   │   │   └── No  → X̄-S 图（n ≥ 9 时 S 比 R 更有效）
│   │   └── No  → 移动极差法？
│   │       ├── Yes → I-MR 图（单值-移动极差）
│   │       └── No  → 使用 X̄-R（合理子组）
│   │
│   └── 数据是否均匀？
│       ├── Yes → 上述控制图
│       └── No  → 分组或使用高级方法
│
└── 计数值（离散数据）
    │
    ├── 缺陷率（每个单元缺陷数）
    │   ├── 子组大小恒定 → c 图
    │   └── 子组大小变化 → u 图
    │
    └── 缺陷品（合格/不合格）
        ├── 子组大小恒定 → np 图
        └── 子组大小变化 → p 图
```

## 控制图类型总览

### 计量值控制图

| 控制图 | 适用场景 | 子组要求 | 优点 | 缺点 |
|--------|---------|---------|------|------|
| **X̄-R 图** | 最常用，通用场景 | n=2~8 | 计算简单，直观 | n>8 时效率降低 |
| **X̄-S 图** | 大子组或自动测量 | n≥9 | 更精确的变差估计 | 计算复杂 |
| **I-MR 图** | 单件生产、慢速过程 | n=1 | 适用于单值数据 | 对非正态敏感 |

### 计数值控制图

| 控制图 | 适用场景 | 数据形式 | 子组要求 |
|--------|---------|---------|---------|
| **p 图** | 不合格率监控 | 合格/不合格判定 | 子组大小可变 |
| **np 图** | 不合格品数监控 | 不合格品计数 | 子组大小恒定 |
| **c 图** | 缺陷数监控 | 缺陷数计数 | 子组大小恒定 |
| **u 图** | 单位缺陷数监控 | 缺陷率 | 子组大小可变 |

## 详细选择指南

### 1. X̄-R 图（均值-极差图）

**最佳场景：** 批量生产，每批次抽取 2~8 件测量

**示例：**
- 机加工零件每天抽 5 件测量直径
- 注塑成型每 2 小时抽 3 件测关键尺寸

**控制限公式：**
- $\bar{\bar{x}} \pm A_2 \times \bar{R}$
- $UCL_R = D_4 \times \bar{R}$
- $LCL_R = D_3 \times \bar{R}$

### 2. X̄-S 图（均值-标准差图）

**最佳场景：** 大子组（n≥9），自动测量设备

**示例：**
- 自动检测设备连续测量 20 件/次
- 化学分析每次测 10 个样本

### 3. I-MR 图（单值-移动极差图）

**最佳场景：** 单件生产、破坏性测试、慢速过程

**示例：**
- 大型设备的最终尺寸检验
- 拉力测试（破坏性）
- 化学反应器的温度监控

**移动极差：** $MR_i = |x_i - x_{i-1}|$

### 4. p 图（不合格率图）

**最佳场景：** 子组大小变化，记录合格/不合格

**示例：**
- 每日来料检验记录不合格率
- 不同批量的出货检验

**注意：** 当子组大小变化超过 ±20% 时，需计算每个子组的单独控制限。

### 5. np 图（不合格品数图）

**最佳场景：** 子组大小恒定

**示例：**
- 每小时固定抽 50 件检验
- 每批固定抽 100 件

### 6. c 图（缺陷数图）

**最佳场景：** 恒定抽样单元，记录缺陷数

**示例：**
- 每块 PCB 的焊点缺陷数
- 每卷布匹的疵点数
- 每台手机的装配缺陷数

### 7. u 图（单位缺陷数图）

**最佳场景：** 抽样单元大小变化

**示例：**
- 每批油漆的外观缺陷（不同批次面积不同）
- 不同长度的电缆缺陷数

## 常见错误

| 错误 | 原因 | 正确做法 |
|------|------|----------|
| 用 p 图处理缺陷数 | 混淆了缺陷品和缺陷 | 缺陷数用 c 图或 u 图 |
| 用 X̄-R 图处理单值数据 | 不了解子组概念 | 使用 I-MR 图 |
| 子组内数据来自不同批次 | 组内变差被夸大 | 确保子组内是同一条件下的连续样本 |
| 忽略数据分布 | 控制图对非正态数据敏感 | 检查正态性，必要时使用高级方法 |
| 过早计算控制限 | 过程不稳定时计算的控制限无效 | 先分析过程稳定性 |

## 选择实例

### 实例 1：注塑车间

**场景：** 注塑机生产手机壳，每 30 分钟抽 5 件测量长度

**选择：** X̄-R 图
- **原因：** 计量值，n=5（在 2~8 范围内）
- **监控：** X̄ 监控均值变化（模具磨损），R 监控变差变化（工艺稳定性）

### 实例 2：电子组装

**场景：** 波峰焊工序，每小时统计 200 件的焊点不良数

**选择：** u 图
- **原因：** 每个 PCB 上焊点数量不同（缺陷机会不同）
- **注意：** 计算单位缺陷数（DPU）

### 实例 3：化学实验室

**场景：** 每天测 1 个样本的粘度

**选择：** I-MR 图
- **原因：** 每个样本成本高，只能测 1 次
- **注意：** 至少需要 20 个点计算控制限
