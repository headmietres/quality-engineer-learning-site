# 学习路径总览

本页面汇总了从零基础入门到进阶提升的完整学习路径。你可以根据自己的当前水平选择适合的路线。

---

## 🟢 初学者路径

适合零基础或刚接触质量工程的学习者，共 **4 步**，预计总耗时 **8-12 周**。

| 步骤 | 内容 | 预计时间 |
|:---:|---|:---:|
| **第一步** | 质量工程概述、QC 七大手法基础、常用质量术语 | 2 周 |
| **第二步** | 统计学基础（均值、标准差、正态分布）、SPC 入门 | 3 周 |
| **第三步** | FMEA 失效模式分析、MSA 测量系统分析基础 | 3 周 |
| **第四步** | ISO 9001 质量管理体系、PPAP 与 8D 报告入门 | 3 周 |

> **建议**：按顺序学习，每完成一步后通过练习巩固。完成此路径后你将对质量工程有全面基础认知。

---

## 🔵 进阶者路径

适合已具备基础质量知识、希望深化技能的学员，共 **3 步**，预计总耗时 **6-9 周**。

| 步骤 | 内容 | 预计时间 |
|:---:|---|:---:|
| **第一步** | 六西格玛方法论（DMAIC）、统计分析深入（假设检验、回归分析） | 3 周 |
| **第二步** | IATF 16949 标准详解、过程审核（VDA 6.3） | 3 周 |
| **第三步** | CQE 认证备考 / 六西格玛绿带/黑带备考 | 3 周 |

> **建议**：如果你已熟悉 QC 七大手法和 SPC 基础，直接从进阶路径开始会效率更高。

---

```
初学者 ──▶ 基础入门 ──▶ 质量工具 ──▶ 体系标准 ──▶ 质检实操 ──▶ 认证考试
                          │
                          └── 进阶者（跳至六西格玛与 IATF 16949）
```
