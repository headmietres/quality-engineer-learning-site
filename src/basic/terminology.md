# 常用术语

以下表格汇总了品质工程领域常用的专业术语及其缩写、中文名称和简要说明，便于快速查阅。

| 缩写 | 全称 | 中文 | 简要说明 |
|------|------|------|----------|
| AQL | Acceptance Quality Limit | 接收质量限 | 抽样检验中可接受的最差质量水平，通常以不合格品百分数表示 |
| CPK | Process Capability Index | 过程能力指数 | 衡量过程满足规格要求的能力，考虑过程中心偏移 |
| DPMO | Defects Per Million Opportunities | 百万机会缺陷数 | 六西格玛中衡量过程绩效的指标，每百万次机会中出现的缺陷数 |
| DPU | Defects Per Unit | 单位缺陷数 | 每个产品单位上平均的缺陷数量，用于衡量缺陷密度 |
| FMEA | Failure Mode and Effects Analysis | 失效模式与影响分析 | 系统性的预防方法，识别潜在失效模式及其影响并采取措施 |
| GRR | Gauge Repeatability and Reproducibility | 量具重复性与再现性 | 衡量测量系统变差中由量具和操作者引起的变差比例 |
| IQC | Incoming Quality Control | 来料质量控制 | 对供应商提供的原材料、零部件进行检验和控制 |
| IPQC | In-Process Quality Control | 过程质量控制 | 生产过程中的质量检验和控制活动 |
| KPI | Key Performance Indicator | 关键绩效指标 | 衡量组织或过程绩效的重要量化指标 |
| LCL | Lower Control Limit | 控制下限 | 控制图中判断过程是否失控的下界限（通常为 $\mu - 3\sigma$） |
| LSL | Lower Specification Limit | 规格下限 | 产品特性允许的最小值，超出即为不合格 |
| MSA | Measurement System Analysis | 测量系统分析 | 评估测量系统误差（偏倚、重复性、再现性等）的系统方法 |
| NCR | Non-Conformance Report | 不符合项报告 | 当产品或过程不符合规定要求时发出的整改通知 |
| OQC | Outgoing Quality Control | 出货质量控制 | 产品出厂前最终检验和控制 |
| PDCA | Plan-Do-Check-Act | 计划-执行-检查-处理 | 持续改进的基本循环方法，又称戴明环 |
| PPM | Parts Per Million | 百万分之一 | 表示不合格品率的单位，1% 的不合格率 = 10,000 PPM |
| PPAP | Production Part Approval Process | 生产件批准程序 | 确保供应商理解并满足客户要求的标准化零件批准流程 |
| QA | Quality Assurance | 质量保证 | 通过体系审核和流程管理，确保产品满足质量要求 |
| QC | Quality Control | 质量控制 | 通过检验和测试，确保产品质量符合标准 |
| QE | Quality Engineering | 质量工程 | 运用工程原理和方法进行质量设计、控制与改进 |
| QFD | Quality Function Deployment | 质量功能展开 | 将客户需求转化为产品设计要求和制造工艺的系统方法 |
| QM | Quality Management | 质量管理 | 制定质量方针、目标和体系，统筹管理质量活动 |
| RPN | Risk Priority Number | 风险优先数 | FMEA 中用于评估风险等级的指标，RPN = 严重度 × 发生频度 × 探测度 |
| SIPOC | Suppliers-Inputs-Process-Outputs-Customers | 供应商-输入-过程-输出-客户 | 用于过程识别的宏观流程图工具，常用于六西格玛项目 |
| SOP | Standard Operating Procedure | 标准作业程序 | 规范操作步骤的书面文件，确保操作的一致性和可重复性 |
| SPC | Statistical Process Control | 统计过程控制 | 应用统计方法监控和控制生产过程，如控制图分析 |
| UCL | Upper Control Limit | 控制上限 | 控制图中判断过程是否失控的上界限（通常为 $\mu + 3\sigma$） |
| USL | Upper Specification Limit | 规格上限 | 产品特性允许的最大值，超出即为不合格 |
| 8D | Eight Disciplines Problem Solving | 八项纪律问题解决法 | 系统性解决问题的 8 步骤方法，常用于客户投诉处理和根本原因分析 |
| Cpk | Process Capability Index (centered) | 过程能力指数 | 衡量过程实际能力（已考虑中心偏移），详见解说（可与 CPK 互换使用） |
| Cp | Process Capability Index (potential) | 潜在过程能力指数 | 衡量过程潜在能力（假设过程对中），$C_p = (USL-LSL)/6\sigma$ |
| COQ | Cost of Quality | 质量成本 | 为达到产品质量要求而发生的全部费用，含预防、鉴定、失败成本 |
| CTQ | Critical to Quality | 关键质量特性 | 对客户满意度至关重要的产品特性，六西格玛项目的关注重点 |
| DoE | Design of Experiments | 实验设计 | 有计划地安排实验因素和水平，以最少的实验次数获取最优参数组合 |
| DMAIC | Define-Measure-Analyze-Improve-Control | 定义-测量-分析-改进-控制 | 六西格玛改进项目的标准流程方法论 |
| NVA | Non-Value Added | 不增值活动 | 在精益生产中指不增加客户价值的活动，应当减少或消除 |
| VSM | Value Stream Mapping | 价值流图 | 用图形化方式展示产品从原材料到成品的全部流程和价值增值过程 |
| APQP | Advanced Product Quality Planning | 产品质量先期策划 | 在产品开发阶段系统化策划质量活动的结构化方法 |
| SCAR | Supplier Corrective Action Request | 供应商纠正措施请求 | 当供应商出现质量问题时，要求其提交原因分析和纠正措施的正式文件 |
| Kaizen | 改善（日语） | 持续改进 | 通过微小而持续的改进积累，实现整体质量水平和效率的提升 |
| Poka-Yoke | 防错法（日语） | 防呆/防错 | 通过设计预防机制，从根本上杜绝操作错误导致的缺陷 |
| Gemba | 现场（日语） | 现场 | 实际发生价值创造的地方，如生产车间、检验区域等 |
| TPM | Total Productive Maintenance | 全员生产维护 | 全员参与的设备维护体系，追求设备零故障和零缺陷 |

---

> **提示**：养成良好的术语使用习惯，在工作中优先使用标准缩写，有助于与同行高效沟通。建议将本表作为随身参考，逐步掌握每个术语的深层含义和应用场景。
