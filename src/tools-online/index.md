## Cp 与 Cpk 的定义

### Cp（过程能力指数）

Cp（Process Capability Index）衡量过程 **潜在能力**，假设过程均值正好对准规格中心。

**公式**：

$$C_p = \frac{USL - LSL}{6\sigma}$$

其中：
- **USL** = 规格上限（Upper Specification Limit）
- **LSL** = 规格下限（Lower Specification Limit）
- **σ** = 过程标准差（用样本标准差 s 或 R̄/d₂ 估计）

### Cpk（过程能力指数，考虑偏移）

Cpk 衡量过程 **实际能力**，考虑了过程均值与规格中心的偏移。

**公式**：

$$C_{pk} = \min\left(\frac{USL - \bar{x}}{3\sigma}, \frac{\bar{x} - LSL}{3\sigma}\right)$$

其中：
- **x̄** = 过程均值
- **USL - x̄** → 上侧能力
- **x̄ - LSL** → 下侧能力
- Cpk 取两者中的 **最小值**

### Cp 与 Cpk 的关系

| 关系 | 含义 |
|------|------|
| Cp = Cpk | 过程均值正好对准规格中心 |
| Cp > Cpk | 过程均值有所偏移（偏移量 = Cp - Cpk） |
| Cpk = 0 | 过程均值正好落在规格界限上 |
| Cpk < 0 | 过程均值已超出规格界限 |

## 计算示例

### 示例数据

某零件长度规格为 **100 ± 1.0 mm**（即 LSL = 99.0, USL = 101.0）。

从生产过程中随机抽取 50 件测量，得到：
- 样本均值 x̄ = **100.3 mm**
- 样本标准差 s = **0.25 mm**

### 计算步骤

**1. 计算 Cp**

$$C_p = \frac{101.0 - 99.0}{6 \times 0.25} = \frac{2.0}{1.5} = 1.33$$

**2. 计算上侧 Cpu**

$$C_{pu} = \frac{101.0 - 100.3}{3 \times 0.25} = \frac{0.7}{0.75} = 0.93$$

**3. 计算下侧 Cpl**

$$C_{pl} = \frac{100.3 - 99.0}{3 \times 0.25} = \frac{1.3}{0.75} = 1.73$$

**4. 计算 Cpk**

$$C_{pk} = \min(0.93, 1.73) = 0.93$$

### 结果解读

| 指数 | 值 | 评价 |
|------|----|------|
| Cp | 1.33 | 潜在能力尚可 |
| Cpk | 0.93 | 实际能力不足 — 均值 **偏向上限** |
| 结论 | — | 需要调整过程均值向 100.0 偏移 |

## 行业标准

| Cpk 值 | 过程能力评价 | 行业要求 |
|--------|-------------|----------|
| ≥ 1.67 | 优秀 | 安全/关键特性（汽车行业） |
| 1.33 ~ 1.66 | 良好 | 一般工业标准 |
| 1.00 ~ 1.32 | 一般 | 需持续改进 |
| < 1.00 | 不足 | 需立即整改 |
