# GRR 在线计算

GRR（Gauge Repeatability & Reproducibility，量具重复性与再现性）是测量系统分析（MSA）的核心内容，用于评估测量系统的变差是否可接受。

> 🔧 **交互计算器即将上线** — 当前页面为概念说明，交互式 GRR 计算工具正在开发中。

## 什么是 GRR

### 基本概念

| 术语 | 英文 | 定义 |
|------|------|------|
| **重复性** (EV) | Repeatability | 同一操作员，用同一量具，多次测量同一零件的 **相同部位** 所得结果的变差 |
| **再现性** (AV) | Reproducibility | 不同操作员，用同一量具，测量同一零件的 **相同部位** 所得结果平均值的变差 |
| **GRR** | Gauge R&R | 重复性与再现性的 **总测量系统变差** |

### 变差关系

```
总变差 (TV) = 零件间变差 (PV) + 测量系统变差 (GRR)
                    │                            │
                    │               ┌─────────────┴─────────────┐
                    │           重复性 (EV)              再现性 (AV)
                    │
              产品的真实差异
```

## %GRR 评价标准

### %GRR 计算

$$\%GRR = \frac{GRR}{TV} \times 100\%$$

其中：
- **GRR** = 测量系统变差（重复性 + 再现性）
- **TV** = 总变差（Total Variation）

### 行业接受标准

| %GRR 值 | 评价 | 行动建议 |
|----------|------|----------|
| ≤ 10% | **优秀** ✅ | 测量系统可接受 |
| 10% ~ 30% | **临界** ⚠️ | 可接受，需根据应用重要性判断，考虑改进 |
| ≥ 30% | **不可接受** ❌ | 测量系统需改进，查找原因并重新分析 |

### 不同行业的 GRR 要求

| 行业 | %GRR 要求 | 说明 |
|------|-----------|------|
| 汽车行业 (AIAG) | ≤ 10% 可接受，10%~30% 有条件接受 | IATF 16949 强制要求 |
| 医疗器械 | ≤ 10% | FDA/GMP 要求 |
| 电子制造 | ≤ 10%（关键特性）| 针对 SMT、精密装配 |
| 一般工业 | ≤ 30% | 非关键特性可放宽 |

## GRR 分析方法

### 1. 均值-极差法 (X̄-R 法)

最常用的 GRR 分析方法，适合现场快速评估。

**分析步骤**：

```
选择3个操作员 × 10个零件 × 每人测量3次 = 90次测量
```

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 选择样本 | 10 个能代表全过程的零件（覆盖整个公差范围） |
| 2 | 编号盲测 | 零件编号隐藏，防止操作员记忆 |
| 3 | 随机顺序 | 每次测量顺序随机化 |
| 4 | 记录数据 | 记录每次测量结果，精确到最小刻度的 1/10 |
| 5 | 计算 GRR | 用公式计算重复性、再现性、%GRR |

### 2. 方差分析法 (ANOVA)

更精确的方法，可分解交互作用（操作员 × 零件）。

**优点**：
- 可估算 **交互作用**（操作员与零件之间的交互效应）
- 更适用于计算机辅助分析
- 可以计算标准差分量、方差分量

**缺点**：
- 计算复杂，需要统计软件支持
- 结果解读需要一定统计基础

## 计算示例

### 示例数据

3 个操作员（A、B、C），10 个零件，每人测量 2 次（简化版，标准为 3 次）。

| 零件 | 操作员 A-1 | 操作员 A-2 | 操作员 B-1 | 操作员 B-2 | 操作员 C-1 | 操作员 C-2 |
|------|-----------|-----------|-----------|-----------|-----------|-----------|
| 1 | 10.1 | 10.2 | 10.3 | 10.2 | 10.1 | 10.2 |
| 2 | 10.5 | 10.4 | 10.5 | 10.6 | 10.4 | 10.5 |
| 3 | 9.8 | 9.9 | 10.0 | 9.9 | 9.9 | 10.0 |
| ... | ... | ... | ... | ... | ... | ... |

### 计算结果（示意）

| 变异来源 | 标准差 | 方差分量 | 占总变差 % |
|----------|--------|----------|-----------|
| 重复性 (EV) | 0.12 | 0.0144 | 15.2% |
| 再现性 (AV) | 0.08 | 0.0064 | 6.8% |
| GRR | 0.14 | 0.0208 | **22.0%** |
| 零件间 (PV) | 0.27 | 0.0729 | 78.0% |
| 总变差 (TV) | 0.31 | 0.0937 | 100% |

**结果解读**：%GRR = 22.0%，在 10%~30% 之间，属于 **临界** 状态。如果该测量用于关键特性，建议改进测量系统。

## GRR 分析通过 / 失败条件

| 条件 | 判定 | 行动 |
|------|------|------|
| %GRR ≤ 10% | ✅ 通过 | 测量系统可接受 |
| 10% < %GRR ≤ 30% | ⚠️ 临界 | 结合应用重要性判断 |
| %GRR > 30% | ❌ 不通过 | 必须改进 |

### 数据分级数 (ndc)

$$ndc = 1.41 \times \frac{PV}{GRR}$$

- **ndc ≥ 5**：测量系统可接受
- **ndc < 5**：测量系统分辨率不足

## GRR 改进方向

### 重复性差（EV 大）的原因

| 原因 | 对策 |
|------|------|
| 量具精度不足 | 更换更高精度量具 |
| 量具磨损/未校准 | 重新校准或维修 |
| 测量方法不统一 | 制定标准化操作程序 (SOP) |
| 零件定位不稳定 | 设计专用检具/治具 |
| 环境因素（温湿度、振动）| 改善测量环境 |

### 再现性差（AV 大）的原因

| 原因 | 对策 |
|------|------|
| 操作员培训不足 | 统一培训并考核 |
| 读数方法不一致 | 统一读数规则 |
| 操作手法差异 | 录制标准操作视频 |
| 主观判断差异（如目视）| 使用标准样板/对比块 |
