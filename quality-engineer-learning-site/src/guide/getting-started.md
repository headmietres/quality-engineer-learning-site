# 快速入门

欢迎开始品质工程师技能学习之旅！本文档将帮助你做好学习前的准备，并提供一份四周速成计划。

---

## 📋 需要什么基础

学习本平台内容，建议具备以下基础：

| 基础要求 | 说明 |
|---------|------|
| **基础数学** | 高中及以上数学水平，理解平均数、百分比等基本概念 |
| **办公软件** | 会使用 Excel 进行基本数据整理和图表制作 |
| **行业背景** | 有制造业或质量管理相关工作经验更佳（非必需） |
| **学习态度** | 保持好奇心和耐心，质量工程需要严谨思维 |

> 如果你的数学基础较薄弱，建议先学习 **[预备知识](/guide/prerequisites)** 中的统计学概念。

## 📅 四周学习计划建议

### 第 1 周：质量工程入门

| 天数 | 学习内容 | 参考链接 |
|:---:|---------|:--------:|
| 第 1-2 天 | 质量工程概述、质量管理发展史 | [基础入门](/basic/) |
| 第 3-5 天 | QC 七大手法（检查表、层别法、柏拉图） | [质量工具](/tools/) |
| 第 6-7 天 | 复习 + 制作 QC 手法笔记 | — |

### 第 2 周：统计基础与 SPC

| 天数 | 学习内容 | 参考链接 |
|:---:|---------|:--------:|
| 第 1-2 天 | 统计学基础（均值、标准差、正态分布） | [预备知识](/guide/prerequisites) |
| 第 3-5 天 | SPC 原理与控制图解读 | [SPC 教程](/tools/spc) |
| 第 6-7 天 | 用 Excel 练习绘制控制图 | — |

### 第 3 周：FMEA 与 MSA

| 天数 | 学习内容 | 参考链接 |
|:---:|---------|:--------:|
| 第 1-3 天 | FMEA 失效模式分析（DFMEA / PFMEA） | [FMEA 教程](/tools/fmea) |
| 第 4-5 天 | MSA 测量系统分析（GRR 等） | [MSA 教程](/tools/msa) |
| 第 6-7 天 | 案例练习：填写 FMEA 表格 | — |

### 第 4 周：体系标准与综合实践

| 天数 | 学习内容 | 参考链接 |
|:---:|---------|:--------:|
| 第 1-3 天 | ISO 9001 质量管理体系核心条款 | [ISO 9001](/standards/) |
| 第 4-5 天 | PPAP 与 8D 报告入门 | [PPAP](/tools/ppap) / [8D](/tools/8d-report) |
| 第 6-7 天 | 完成一个综合质量改进案例 | — |
