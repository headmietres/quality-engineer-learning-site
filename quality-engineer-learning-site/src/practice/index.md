# 来料检验 (IQC)

IQC（Incoming Quality Control，来料质量控制）是质量检验的第一道关口，负责对供应商交付的原材料、零部件进行检验和验证，确保只有合格物料进入生产线。

## 来料检验流程

```
供应商送货 → 仓库暂收 → 通知IQC → 抽样/全检 → 判定 → 
标识状态 → 入库/退货 → 记录归档
```

### 各环节说明

| 步骤 | 职责 | 说明 |
|------|------|------|
| 仓库暂收 | 仓库 | 核对送货单与采购订单，数量清点，物料暂存待检区 |
| 通知IQC | 仓库/系统 | 通知 IQC 检验员到现场抽样检验 |
| 抽样检验 | IQC | 按抽样计划抽取样本，进行外观/尺寸/性能检测 |
| 结果判定 | IQC | 合格（Accept）、让步接收（Concession）、退货（Reject） |
| 标识状态 | IQC | 贴合格/不合格/待处理标签，分区存放 |
| 记录归档 | IQC | 填写检验报告，录入系统，保存追溯记录 |

## 抽样计划与 AQL 标准

### GB/T 2828.1 计数抽样检验

中国国家标准 **GB/T 2828.1**（等同 ISO 2859-1）是来料检验中最常用的计数调整型抽样方案。

### 关键术语

| 术语 | 含义 |
|------|------|
| **AQL** (Acceptable Quality Limit) | 可接受质量水平，即合格批中允许的最大不合格品百分数 |
| **N** | 批量（Lot Size） |
| **n** | 样本量（Sample Size） |
| **Ac** | 接收数（Acceptance Number）—— 样本中允许的最大不合格品数 |
| **Re** | 拒收数（Rejection Number）—— 样本中不合格品数达到此数则拒收 |
| **正常/加严/放宽** | 根据历史质量水平切换的三种检验严格度 |

### 抽样类型

| 类型 | 说明 | 适用场景 |
|------|------|----------|
| 一次抽样 | 抽一个样本即判定 | 日常来料检验 |
| 二次抽样 | 抽第一个样本，若结果在灰色区域则抽第二个 | 需要降低平均样本量的场景 |
| 多次抽样 | 最多七次抽样 | 检验成本高、批量大的场景 |

### 检索步骤

1. 确定 **AQL 值**（行业惯例：关键特性 AQL=0，重要特性 AQL=0.65~1.0，一般特性 AQL=1.5~4.0）
2. 查 **批量范围 N**，确定**字码（Code Letter）**
3. 根据字码和 AQL 查 **n、Ac、Re**
4. 执行抽样并判定

## 供应商质量管理

### 供应商导入流程

```
供应商寻源 → 初评 → 样品认证 → 小批量试产 → 
现场审核 → 合格供应商名录 → 定期绩效评价
```

### 供应商质量评分（QCDS 模型）

| 维度 | 权重 | 指标 |
|------|------|------|
| **Q**uality（质量） | 40% | 来料批次合格率、PPM、质量问题响应速度 |
| **C**ost（成本） | 25% | 价格竞争力、降本提案 |
| **D**elivery（交付） | 25% | 准时交付率、交付周期 |
| **S**ervice（服务） | 10% | 配合度、技术支援能力 |

### 供应商分级管理

- **A 级供应商（优秀）**：减少检验频次，优先付款
- **B 级供应商（合格）**：正常检验频次
- **C 级供应商（待改进）**：加严检验，限期改善
- **D 级供应商（淘汰）**：启动淘汰流程

## 来料不合格处理流程

```
IQC判定不合格 → 标识隔离 → 填写NCR → 通知采购/SQE → 
MRB评审 → 四种处置方式之一 → 记录关闭
```

### MRB 评审四种处置方式

| 处置方式 | 定义 | 适用条件 |
|----------|------|----------|
| **让步接收** (Concession) | 经评审后让步使用，不影响功能 | 不影响装配或性能，获得客户同意 |
| **返工** (Rework) | 全检挑选或修复后复检 | 经返工后可达到合格标准 |
| **退货** (Reject/Return) | 退回供应商 | 严重不合格，无使用价值 |
| **报废** (Scrap) | 按报废流程处置 | 无法返工且无让步价值的物料 |

## 检验记录管理

### 检验记录内容

| 记录项 | 说明 |
|--------|------|
| 供应商信息 | 供应商名称、代码 |
| 物料信息 | 料号、批号、数量、生产日期 |
| 检验标准 | AQL 值、检验规格/图纸版本 |
| 检验数据 | 外观、尺寸、性能等实测值 |
| 判定结论 | 合格/让步/退货 |
| 签名与日期 | 检验员、审核人签名 |

### 记录保存要求

- **保存期限**：至少 **3 年**（汽车行业建议 **15 年**）
- **保存形式**：纸质归档 + 电子系统备份
- **可追溯性**：按批次号/Lot No. 可追溯至供应商原始记录

---

> **核心原则**：来料检验不是目的，而是手段。通过 IQC 数据驱动供应商质量改进，逐步实现免检才是质量管理的最终目标。
