# 出货检验 (OQC)

OQC（Outgoing Quality Control，出货质量控制）是产品出厂前的最后一道质量把关，确保交付给客户的产品符合规格要求和质量标准。

## 出货检验流程

```
生产完工 → 入库申请 → OQC抽样检验 → 
判定：合格→放行出货 / 不合格→退回处理 → 
签发COA/检验报告 → 出货
```

### 各环节详细说明

| 步骤 | 责任部门 | 操作内容 |
|------|----------|----------|
| 生产完工 | 生产部 | 完成生产，产品入库待检区，提交《完工检验申请单》 |
| OQC 抽样 | OQC | 按抽样计划从待检批中随机抽取样品 |
| 实施检验 | OQC | 外观、尺寸、功能、包装、标识逐项检验 |
| 综合判定 | OQC | 根据检验结果判定批次合格/不合格 |
| 标识放行 | OQC/仓库 | 合格品贴合格标签，办理入库/出货手续 |
| 报告签发 | OQC | 出具检验报告/COA |

## 成品抽样标准

### 抽样方案选择

| 客户要求 | 抽样标准 | 说明 |
|----------|----------|------|
| 无特殊要求 | GB/T 2828.1 (AQL) | 一般检验水平 II |
| 客户指定 AQL | 按客户 AQL 值 | 常见 AQL=0.65、1.0、1.5、2.5 |
| 汽车行业 | AIAG CQI 或客户特殊要求 | 零缺陷要求，通常 AQL=0 |
| 批量极小（≤50） | 全检或加严抽样 | 尽可能全检 |
| 破坏性检验 | 特殊抽样方案 | 最小样本量，统计推算 |

### 检验水平选择

| 检验水平 | 相对判别力 | 样本量 | 适用场景 |
|----------|-----------|--------|----------|
| **特殊水平** (S-1~S-4) | 最低 | 最小 | 破坏性检验、高成本检验 |
| **一般水平 I** | 较低 | 较小 | 低风险产品、历史质量稳定的产品 |
| **一般水平 II** | 正常 | 适中 | **默认使用**，大多数产品 |
| **一般水平 III** | 较高 | 较大 | 高风险产品、新产品、新供应商 |

## 包装检验

### 包装检验内容

| 检验项目 | 标准要求 | 检验方法 |
|----------|----------|----------|
| 包装材质 | 符合包装规范，无破损变形 | 目视检查 |
| 内包装 | 防静电/防潮/防震措施到位 | 目视+测量 |
| 外包装 | 唛头清晰，数量准确，封箱牢固 | 目视检查 |
| 标签标识 | 产品名称、型号、数量、生产日期、批次号、MADE IN CHINA | 目视核对 |
| 包装数量 | 与装箱单一致 | 逐一清点或称重核验 |
| 环保标识 | RoHS/REACH 等环保合规标识 | 核对资质文件 |

### 常见包装缺陷

- 纸箱受潮/变形
- 标签信息错误或脱落
- 内外标识不一致
- 防护材料缺失（气泡膜、隔板等）
- 混装不同批次/规格产品
- 包装数量不足或超出

## 检验报告与 COA

### COA（Certificate of Analysis）内容

COA（分析/检验证书）是 OQC 出具给客户的官方质量证明文件，通常包含以下内容：

| 项目 | 内容 |
|------|------|
| **基本信息** | 供应商名称、客户名称、产品名称、型号/规格 |
| **批次信息** | 生产批号、生产日期、检验日期、数量 |
| **检验标准** | 规格公差、检验水平、AQL 值 |
| **检验结果** | 外观、尺寸、性能等实测值及判定 |
| **仪器信息** | 使用量具名称、编号、校准有效期 |
| **综合结论** | 合格/不合格 |
| **授权签名** | 检验员、审核人、质量经理签名/印章 |

### COA 管理要点

1. **真实性**：所有数据应可追溯至原始检验记录
2. **完整性**：不遗漏任何客户要求的检测项目
3. **及时性**：出货前完成签发，随货同行
4. **存档**：纸质+电子存档，保存期限按客户要求（一般≥3年）
5. **防伪**：加盖公司质量专用章或防伪标识

## 客户投诉与退货处理

```
收到客诉 → 登记客诉台账 → 
紧急处置（挑选/换货/退货） → 
根本原因分析 → 
纠正与预防措施 → 
措施验证 → 
客诉关闭 → 横向展开
```

### 客户投诉处理原则

| 原则 | 说明 |
|------|------|
| **快速响应** | 24 小时内回复客户，48 小时内给出初步方案 |
| **及时沟通** | 持续向客户通报处理进展 |
| **根本解决** | 不满足于表面整改，必须找到根本原因 |
| **防止再发** | 整改措施横向展开到同类产品/工序 |
| **记录完整** | 每个环节留痕，形成完整 8D 报告 |

### 退货处理流程

```
客户退货 → 仓库登记接收 → 退货隔离区 → 
退货检验确认 → 填写NCR → 
MRB评审 → 返工/报废/让步 → 
纠正措施 → 关闭
```

### 退货类型与责任判定

| 退货类型 | 判定依据 | 责任方 | 费用承担 |
|----------|----------|--------|----------|
| 质量缺陷 | OQC 漏检/未检出 | 我司 | 我方承担退货运费+损失 |
| 包装破损 | 运输原因 | 物流公司 | 物流公司承担 |
| 规格不符 | 设计变更未通知 | 销售/研发 | 内部流程整改 |
| 客户使用不当 | 使用条件超出规格 | 客户 | 客户承担（做好客情维护） |
| 争议件 | 双方各执一词 | 协商 | 第三方检测鉴定 |

### 关键指标

| 指标 | 计算公式 | 目标 |
|------|----------|------|
| **客诉率** | 客诉批次数/总出货批次数 × 100% | ≤ 1% |
| **退货率** | 退货数量/总出货数量 × 100% | ≤ 0.5% |
| **客诉关闭率** | 已关闭客诉数/总客诉数 × 100% | ≥ 95% |
| **客诉平均关闭天数** | ∑客诉关闭天数/客诉总数 | ≤ 15 天 |

---

> **核心原则**：OQC 不是终点，而是客户满意度的起点。每一次出货都是品质承诺的兑现。
